# ChatGPT Review - Opus Nexus Design System

Status: design-system handoff for Codex implementation.

This folder captures the current Opus Nexus visual direction.

Direction:
- Premium personal assistant / personal command center
- Roman inscriptions, sharp geometric, authoritative
- Dark UI with restrained gold authority accent

Core rules:
1. Navigation and Utility tiles use near-square radius: 8-10px.
2. Gold is a brand authority accent, not an all-app color wash.
3. Keep gold around 10-15 percent visual weight.
4. Health primary uses Heart.
5. Vita uses Guardian Shield: shield means protection, pulse means vital signal.
6. Fiber uses Fiber Leaf.
7. Icons use controlled 2px-ish stroke and avoid emoji/playful style.
